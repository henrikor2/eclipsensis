Script Wizard

Installer Presentation

This page is used to customize how the installer presents itself to the user.

Show License
Option to display a license page to the user, which has to be accepted by the user before proceeding. ← Back
License Content File
Location of the file with the license content. This may be a plain text (*.txt) or Rich Text (*.rtf) file. ← Back
License Button Type
Type of button for the user to accept the license. The current options are a standard push button, a check box or Yes/No radio buttons. ← Back
Show Splash Screen
Option to display a splash screen when starting the installer. ← Back
Splash Image File
Location of image file for the splash screen. This must be a Windows Bitmap file (*.bmp). ← Back
Splash Sound File
Optional location of a sound file to be played while displaying the splash screen. This must be a WAVE PCM audio file (*.wav). ← Back
Display Delay
Length of time to display the splash screen, in milliseconds. ← Back
Fade-in Delay (Windows 2000 or better)
Optional length of time to fade into the splash screen, in milliseconds. ← Back
Fade-out Delay (Windows 2000 or better)
Optional length of time to fade out of the splash screen, in milliseconds. ← Back
Splash Preview
Preview the splash screen. Fade-in/fade-out effects are only displayed on Windows 2000 or better. ← Back
Show Background
Option to create a gradient background window for the installer. ← Back
Background Image File
Optional location of image file to be displayed centered on the background. This must be a Windows Bitmap file (*.bmp). ← Back
Background Sound File
Optional location of a sound file to be played in a loop while displaying the background. This must be a WAVE pcm audio file (*.wav). ← Back
Top Color
Top color for the gradient. ← Back
Bottom Color
Bottom color for the gradient. ← Back
Text Color
Color for text displayed on the background. ← Back
Background Preview
Preview the background window. ← Back
