Installation

System Requirements

  1. One of the following operating systems:
    • Windows NT 4.0
    • Windows 2000
    • Windows XP
    • Windows Server 2003
    • Windows Vista
    • Windows Server 2008
    • Windows 7
  2. One of the following processor architectures:
    • x86 (32-bit)
    • x64 (64-bit)
  3. NSIS 2.0 or better. http://nsis.sourceforge.net
  4. Eclipse 3.5 (Galileo) or better.
  5. JRE 5.0 or better. EclipseNSIS has been certified with the Sun JRE.

Getting EclipseNSIS

There are two ways to get and install EclipseNSIS:
  1. (Recommended) Use the Eclipse Software Updates and Add-ons utility or the classic Eclipse Update Manager to connect to http://eclipsensis.sf.net/update and download the latest release.
  2. Download directly from SourceForge and unzip into the Eclipse folder.
