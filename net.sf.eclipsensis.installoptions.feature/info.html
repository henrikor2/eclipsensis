EclipseNSIS InstallOptions Editor Information

EclipseNSIS InstallOptions Editor is an add-on plugin to the EclipseNSIS plugin, which enables designing and editing Nullsoft Installer (NSIS) InstallOptions scripts.
It requires that the Eclipse Graphical Editor Framework (GEF) 3.1 is installed.
