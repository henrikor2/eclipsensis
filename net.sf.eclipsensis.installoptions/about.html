About EclipseNSIS InstallOptions Editor

EclipseNSIS InstallOptions Editor is a plugin for the Eclipse platform which enables creating and editing InstallOptions dialogs used by the Nullsoft Installer (NSIS).
It is built using the Eclipse Graphical Editor Framework (GEF).
To install or update, visit EclipseNSIS Update Site in the Update Manager.
